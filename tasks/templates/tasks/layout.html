About this site
Note is a free online tool that you can use to write down what you have in mind.
Each Note entry can be viewed by visiting that entry’s page. Notice that the name of the requested page (HTML) is specified in the route /note/HTML. Recognize too, that the page’s content must just be HTML that your browser renders.
In practice, it would start to get tedious if every page on Note had to be written in HTML. Instead, it can be helpful to store Note entries using a lighter-weight human-friendly markup language. Note happens to use a markup language called Markdown.
Read through GitHub’s Markdown guide to get an understanding for how Markdown’s syntax works. Pay attention in particular to what Markdown syntax looks like for headings, bold text, links, and lists.
By having one Markdown file represent each Note entry, we can make our entries more human-friendly to write and edit. When a user views our Note entry, though, we’ll need to convert that Markdown into HTML before displaying it to the user.
